Add public method `BacktestResult run(BacktestConfig config, Map<String, List<HistoricalBar>> barsBySymbol)`. Implementation: (1) Pre-index bars into `Map<String, Map<LocalDate, HistoricalBar>> index` for O(1) lookup. (2) Build sorted unique `List<LocalDate> tradingDays` from all bar dates across all symbols. (3) Initialize `double cash = config.getStartingBalance()`, `Map<String, double[]> openPositions` where each entry is `{shares, entryPrice}`, per-symbol `Map<String, List<Double>> priceHistory` and `Map<String, List<Double>> volumeHistory`. (4) For each date in tradingDays: for each symbol in config.getSymbols(), look up bar; if absent skip; append close to `priceHistory.get(symbol)` and volume to `volumeHistory.get(symbol)` (initializing lists lazily if needed); if list size < 30 skip signal evaluation; otherwise call `indicatorEngine.evaluateAll(prices, volumes, close)`; count `buys` (direction == BUY) and `sells` (direction == SELL). If symbol has an open position and (sells >= 2 or close < openPositions.get(symbol)[1] * 0.85): execute close — shares = (int)pos[0], cash += shares * close - feeCalc.calculateFee(shares), totalTrades++, wins++ if close > pos[1] else losses++, remove position. Else if symbol has no open position and buys >= 2: shares = feeCalc.maxShares(cash, close); if shares > 0 then cash -= shares * close + feeCalc.calculateFee(shares), openPositions.put(symbol, new double[]{shares, close}). After processing all symbols for the day, compute portfolioValue = cash + sum(openPositions: pos -> pos[0] * latestClose[symbol]) using a helper that gets the last bar price for each open symbol on this date or falls back to the last seen price; add `new BacktestDataPoint(date, portfolioValue)` to equityCurve. (5) After all days, close any remaining open positions at their last recorded price (same close logic, no trade outcome recorded). (6) Compute totalReturnPct = equityCurve is empty ? 0.0 : (equityCurve.get(last).getPortfolioValue() - config.getStartingBalance()) / config.getStartingBalance() * 100. (7) Compute maxDrawdownPct: iterate equityCurve, track running peak, maxDrawdown = max((peak - v) / peak * 100). (8) Return new BacktestResult(equityCurve, totalReturnPct, maxDrawdownPct, wins, losses, totalTrades). Required imports: `com.tradingapp.data.HistoricalBar`, `java.time.LocalDate`, `java.util.*`.
